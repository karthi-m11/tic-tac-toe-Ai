# Tic-Tac-Toe AI

## About
A simple Tic-Tac-Toe game where a human player plays against an AI.
The AI uses a basic Minimax algorithm to select the best move.

## How to Play
- You are X, AI is O
- Enter numbers 0-8 to place your X
- AI will automatically make its move
- Game ends when someone wins or board is full
